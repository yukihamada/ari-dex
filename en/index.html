Why This Works

なぜ ARI は成功するのか

既存プレイヤーが解決できていない3つの構造的問題に、正しいタイミングで正しい解を提示する。

Core Thesis

DEXのスポット取引量は2024年に$1.76Tに達し（CoinGecko）、DEX/CEXスポット比率は約20%まで拡大（The Block, 2025）。 しかし累計$3B以上がMEVとして搾取され（Flashbots, 2025）、UXの壁から取引の約80%は依然としてCEXに留まっている。 根本原因は「注文の順序依存性」と「チェーンの分断」。 インテントベース + 暗号化バッチオークションはこれを構造的に解決する最も有力なアーキテクチャであり、今まさにインフラが実用段階に達した。

Reason 1 — Problem

MEV問題は設計レベルで壊れている

Uniswapの注文は公開メモリプールを通過する。フロントランニングは原理的に防げない。Hookやプライベートメモリプールは「パッチ」。

ARI: 閾値暗号で注文を秘匿 → バッチ確定まで誰も中身を見れない → フロントランニングが物理的に不可能
Reason 2 — Opportunity

流動性の分断が勝者総取りの機会を生む

20以上のチェーンに流動性が分散。統一された体験を提供するプレイヤーはまだいない。「全部入り」は誰もやっていない。

ARI: インテントレイヤーがチェーンを抽象化。ユーザーは「ETHをUSDCに」と言うだけ。ソルバーが全チェーンから最適ルートを発見。
Reason 3 — Unfair Advantage

AIが開発コストの常識を破壊する

AIがコード生成を加速するが、暗号プロトコルのボトルネックは「数式の検証とセキュリティ担保」。開発費を圧縮した分を、外部監査・フォーマル検証・バグバウンティに全額投入。

ARI: AIでコード生成を10倍加速。ただし暗号・ZKの検証には外部専門家を必須配置。浮いた資金はセキュリティ監査（3社）とバグバウンティ（$10M）に全振り。

Core Innovation

Intent Engine

ユーザーは「意図」だけ伝える。最適な実行はプロトコルが保証する。

1

Intent Submission

「1 ETHを最良価格でUSDCに交換」— EIP-712準拠の署名付きインテントを送信

2

Threshold Encryption

閾値暗号（t-of-n BLS）で暗号化。バッチ確定まで誰も内容を閲覧不可

3

Solver Auction

ソルバーがブラインド入札で最良ルートを競争。CEX流動性も参加可能

4

Batch Decryption & Matching

250msバッチ終了後に一括復号。同一価格で約定（均一価格オークション）

5

Settlement

ZK証明付きでオンチェーン確定。トークンがウォレットに直接到着

Security

MEV Protection

累計$3B以上が搾取されるMEV問題を、パッチではなくアーキテクチャで解決

🔒 Encrypted Mempool

閾値暗号（t-of-n BLS）で注文内容を秘匿。バリデータの2/3が復号に参加しない限り不可視。サーチャーは何も見えない。

⚖ Batch Auction

250msバッチ内の全注文を均一価格で約定。注文の順序に意味がない = 先行者利益ゼロ。

💰 MEV Rebate

残余MEV（統計的アービトラージ等）の80%をユーザーに還元。残り20%がプロトコル収益。

Vesting Schedule — 売り抜け防止

「上場直後に運営とVCが売り抜けて暴落」を構造的に不可能にする。チームは4年フルコミット。

Public Saleなしなし即時TGE時に全量流通

Value Accrual — 「配当」なしで価値を上げる

日本の金商法リスク: プロトコル収益をトークンホルダーに直接配当すると「集団投資スキーム」=有価証券とみなされ、完全に身動きが取れなくなる。
ARI の回避策: 収益を「ユーティリティ（実用性）」と「トークンの希少性向上」に転換する。

Work Token Model

ソルバーとしてネットワークに参加しスプレッド利益を得るには、最低100,000 $ARIのステーキングが必須。

効果: 取引量増 → ソルバー参入増 → $ARI買い圧 → 市場からロック。配当ではなく「仕事をするための道具」としてのユーティリティ。日本法上は実用トークン。

veARI Governance

$ARIを長期ロック → veARI（投票権）を取得。ロック期間に応じて投票力が増加。

veARI保有者の実用的メリット:
• LP報酬のAPRブースト（最大2.5倍）
• API Proプラン無料（通常$ARI stake 1,000+）
• ガバナンス投票権（手数料率、新チェーン追加、トレジャリー使途）
• エコシステムグラント審査への参加権

Buyback & Make

プロトコル収益（MEV残余20% + 取引手数料）は配当ではなく以下に使用:

1. DAOトレジャリーの拡大（保険基金）
2. 市場からの$ARI買い上げ → エコシステムファンドに還元
= 配当ではないため金商法をクリア
